# tg-bot-prototype
easy tg-bot for customer acquisition or test task for developers

ТГ-бот для помощи юным разработчикам

ИНСТРУКЦИЯ:
1. Начните с установки библиотеки pip install pyTelegramBotAPI
2. Импортируйте модуль import telebot
3. Ознакомтесь с библиотекой на pypi.org (pyTelegramBotAPI),
чтобы под чистую не копипастить код и добавить функции, которые будут нужны вашему боту.

Успехов юный прогер!
